
# 📊 Sales Performance Analysis | Data Analytics Project

## 📌 Project Overview

This project focuses on analyzing Superstore sales data to evaluate sales performance across different regions, product categories, customer segments, and time periods. The analysis includes data cleaning, exploratory data analysis (EDA), data visualization, dashboard creation, and business recommendations to support data-driven decision-making.

This project was completed as part of the **InternSpark Data Analyst Internship**.

---

## 🎯 Objectives

* Clean and prepare the sales dataset.
* Analyze sales performance across regions and categories.
* Identify top-performing products and customer segments.
* Explore monthly and yearly sales trends.
* Build an interactive dashboard.
* Generate actionable business insights and recommendations.
